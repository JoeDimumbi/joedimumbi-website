Password Generator

Project Information

Category: Security Tool

Technologies: Python

Project Date: 13 February 2020

Project repo: GitHub

View Project

A secure and customizable password generator that allows users to create strong, random passwords with various character sets. One of my responsibilities was onboarding and offboarding agents (setting up their devices + creating app logins). For every device a new password was needed, hence the idea of random password generator which I had already implemented on google sheet but wanted to have it automated somehow and later add more features.
